# STRIKON ⚽

**The Ultimate Premium Football Hub**

STRIKON is a highly-responsive, ultra-premium web application built for football enthusiasts. It delivers real-time global fixtures, dynamic match updates, and exclusive tournament data (including the FIFA World Cup) wrapped in a stunning esports-style glassmorphism UI.

### 🌟 Features
- **Live Global Fixtures:** Powered by API-Sports to fetch and display the most anticipated upcoming matches worldwide.
- **Dynamic Match Hero:** An auto-sliding, data-driven Hero section that elegantly highlights the biggest global events.
- **Secure Authentication:** Integrated with Google Firebase for secure, real-time user accounts and sessions (Email & Google OAuth).
- **Premium UI/UX:** Built with Next.js, Tailwind CSS, and Lucide Icons to deliver lightning-fast micro-animations, glowing gradients, and a flawless mobile experience.

### 🛠️ Tech Stack
- **Frontend:** Next.js (App Router), React, Tailwind CSS
- **Backend/Auth:** Google Firebase (Authentication & Context)
- **Data:** API-Football (v3)
